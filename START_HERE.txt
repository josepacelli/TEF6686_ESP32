================================================================================
                       TROUBLESHOOTING QUICK TIPS
================================================================================

"Error: duplicate case in switch"
  → Your old function was not completely deleted
  → Try again: delete entire old function, then paste new one

"Some themes appear black"
  → Color values may have been corrupted during copy
  → Verify all colors are in format 0xXXXX (4 hex digits)
  → Try pasting again carefully

"Theme names do not show"
  → Verify getThemeName() was actually replaced
  → Check that UI code calls getThemeName(themeSet)

"Will not compile - file too large"
  → ESP32 flash may be full
  → Disable other features or use larger ESP32

For more troubleshooting, see INTEGRATION_GUIDE.md
