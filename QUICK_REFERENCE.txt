================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: "error: duplicate case in switch"
CAUSE: Old function not completely removed
FIX: Verify you deleted the entire old function before pasting new code

PROBLEM: Some themes appear black
CAUSE: Color hex values corrupted during copy
FIX: Verify all colors are in 0xXXXX format (4 hex digits)

PROBLEM: Theme names don't update
CAUSE: getThemeName() not being called
FIX: Check UI rendering code is calling getThemeName(themeSet)

PROBLEM: Firmware too large (won't fit)
CAUSE: Code size exceeded ESP32 flash
FIX: Disable other features or use ESP32 with larger flash

PROBLEM: Compilation succeeds but no visible change
CAUSE: Old firmware still running on device
FIX: Full power reset, or erase flash before reflashing
